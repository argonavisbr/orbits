Objetos sistema solar

Este programa pesquisa em uma base de dados XML de objetos do sistema solar contendo 5024 objetos obtidos da base do Jet Propulsion Laboratory da NASA. Os dados foram atualizados em dezembro de 2010.

Use o formulário abaixo para alterar a aparência dos resultados e filtrar os resultados por tipo de objeto, diâmetro e órbita máxima e mínima, e mostrar menos objetos.

Limitações:

  • Esta é uma aplicação experimental. Não há validação nos campos abaixo.
  • A reordenação dos dados da tabela só é possível quando as luas não são exibidas.
