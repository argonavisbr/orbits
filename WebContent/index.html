Como os dados foram extraídos

Os dados do Sol e oito planetas principais foram inseridos manualmente.

Os dados dos satélites, asteróides, objetos trans-netunianos, centauros e cometas foram extraídos da base do Jet Propulsion Laboratory (JPL) da NASA através das ferramentas Commons HTTPClient e JTidy usadas para converter os resultados das pesquisas (tabelas HTML) em XML, extraíndo os dados selecionados e convertendo-os nos formatos e unidades usadas na aplicação (através de Java e várias folhas de estilo XSLT). A última atualização foi em dezembro de 2010.

Os programas usados serão disponibilizados aqui em breve.

Como os gráficos e tabelas foram gerados

Os gráficos SVG de órbitas são gerados através de folha de estilos XSLT, que é processada pelo Saxon 9.0 embutido em um servlet Java. Todos os arquivos utilizados na transformação estão listados abaixo (TODO: svn links):

  • Folha de estilos XSLT 1.0 para gerar animação com órbitas (versão 0.2): sol_orbitas_svg_2.xsl
  • Dependências: valores_de_referencia.xsl, parametros_sol_orbitas.xsl, defaults.xsl, geometria.xsl, matematica.xsl
  • Folha de estilos XSLT para gerar tabelas (versão 0.1): sol_tabelas_html.xsl
  • Dependências: parametros_sol_tabelas.xsl
  • Aplicação Web Orbitas: web.xml, TransformacaoServlet.java, TransformacaoSVG.java
  • Aplicação Web Tabelas: web.xml, TransformacaoHTML.java

A folha de estilos usa XSLT 1.0 e XPath 1.0 sem extensões (e não versões mais novas) porque é usada em um treinamento que aborda essas versões separadamente (por motivos práticos e suporte).
